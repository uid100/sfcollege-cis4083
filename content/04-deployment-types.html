Cloud Deployment Types

    Four cloud deployment models are widely recognized: public clouds, private clouds:
  • Public Cloud: Services are delivered over the public internet and shared across organizations.
  • Private Cloud: Cloud infrastructure is operated solely for a single organization, either on-premises or hosted by a third party.
  • Community Cloud: Infrastructure is shared among several organizations with common concerns, such as security or compliance requirements.
  • Hybrid Cloud: A combination of two or more cloud deployment models, allowing data and applications to be shared between them.
A cloud deployment model is defined according to where the infrastructure for the deployment resides and who has control over that infrastructure.
