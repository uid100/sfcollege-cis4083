Screenshots

Create the account in each of the cloud service providers as described.

Next, login to your account and take a screenshot showing the main dashboard and your active account.

 

Here is an example logged into the Azure Portal. This was using the Google Chrome browser on a Macbook Pro

 

And another, this time also using Chrome on a Windows 11 notebook.

 

Note: when you take your screenshots, size the window down to something reasonable and clip only the relevant content.

Do not use Print Screen.

With MacOS, use shift-command-4 and click and drag the crosshair across the required area.

Using Windows, use the Snipping Tool

 

Save your screenshots as common image formats, I prefer jpeg (.jpg), Portable Network Graphic (.png) is also fine. It is the default from the Mac capture.

Upload both of these files to Canvas. If you are having difficulty submitting both files, you may zip them into a single (not individual) zip archive file. Do not use tar or cpio or 7zip or any other compression tool. These image files are already compact formats, so uploading the original image files is preferred. Do not attach them to the Canvas Comment window. I will not see the attachments.
