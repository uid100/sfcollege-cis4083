Exercise 1 - Backend API

The objective of this exercise is to build and deploy a web application using a backend API.

Create a Word document and include screenshots and descriptions of the steps you took to:

  1. Create and configure an Azure Function App with a custom API
  2. Demonstrate the working API using a tool such as curl
  3. Deploy a web application that consumes the API
  4. Test and demonstrate the functionality of the web application

Submit the single Word document to the assignment submission link.
